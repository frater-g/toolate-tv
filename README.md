# The Too Late Show Website

Official website for The Too Late Show - "No War but the Psywar"

## Design

- **Aesthetic:** Dark metallic industrial (H.R. Giger / Warhammer 40k inspired)
- **Typography:** Courier New lowercase primary
- **Colors:** Black, dark gray, chrome, silver - fully sanitized (no rust/red)
- **Vibe:** Clean, sinister, gothic, elegant

## Tech Stack

- Next.js 14 (React framework)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Pages

- Home: Hero + featured content
- About: Show concept and Dr. Belial Crow lore
- Episodes: Show archive (placeholder)
- Blog: Articles and commentary (placeholder)
- Media: Image/video gallery (placeholder)
- Links: Social media hub (placeholder)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

This site is designed to deploy to Vercel with zero configuration.
